# workers

Python 3.12 pipeline workers (managed with `uv`). Real workers — slicer,
transcriber, diarizer, stitcher, extractor — land starting Phase 1; see
[docs/plan.md](../docs/plan.md) and [docs/architecture.md](../docs/architecture.md#the-racing-engine).

Setup once `uv` is installed: `uv sync` from this directory.
